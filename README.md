
# Zentech 🛒
### Plataforma de E-commerce desenvolvida com Vue.js

Zentech é um projeto de plataforma de e-commerce desenvolvido com o objetivo de aplicar na prática conceitos modernos de desenvolvimento web utilizando o ecossistema JavaScript.

A aplicação simula o funcionamento de uma loja online de produtos tecnológicos, permitindo a navegação por produtos e a interação com uma interface moderna e organizada.

O projeto está sendo desenvolvido progressivamente como parte das atividades de aprendizado em desenvolvimento web.

---

## 🎯 Objetivo do Projeto

O principal objetivo do projeto é praticar o desenvolvimento de aplicações web modernas utilizando frameworks e ferramentas amplamente utilizadas no mercado.

Além disso, o projeto busca:

- Aplicar conceitos de desenvolvimento front-end
- Trabalhar com componentes reutilizáveis
- Utilizar bibliotecas modernas de interface
- Praticar organização e estruturação de projetos
- Utilizar controle de versão com Git e GitHub

---

## 🚀 Tecnologias Utilizadas

O projeto utiliza tecnologias modernas do ecossistema JavaScript:

**Linguagens**
- TypeScript
- JavaScript
- HTML

**Frameworks e Bibliotecas**
- Vue.js
- PrimeVue
- Tailwind CSS

**Ferramentas e Ambiente de Desenvolvimento**
- Node.js
- Vite (ferramenta de build)
- npm (gerenciamento de dependências)
- Git (controle de versão)
- GitHub (repositório do projeto)
- Visual Studio Code

---

## 🧩 Estrutura do Projeto

A aplicação é estruturada utilizando o framework Vue.js, permitindo a organização da interface em componentes reutilizáveis.

Principais características da estrutura:

- organização modular de arquivos
- separação entre estrutura, estilo e lógica da aplicação
- utilização de componentes Vue
- estilização moderna com Tailwind CSS
- componentes de interface com PrimeVue

Essa organização facilita a manutenção e evolução do projeto.
